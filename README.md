# 桌面行程表

這是一個免安裝的本機桌面行程表，直接雙擊 `index.html` 即可使用。

## 已包含功能

- 每日行程輸入
- 日 / 週 / 月檢視切換
- 任務完成勾選
- 優先順序：高 / 中 / 低
- 分類與顏色標記：工作、家庭、學習、運動、重要事項
- 自訂分類與分類顏色
- 自訂畫面文字：例如主標題、按鈕、區塊標題、欄位名稱
- 自訂日檢視顯示時間範圍
- 子任務清單
- 一鍵複製行程到明天
- 行程置頂
- 逾時未完成行程自動標紅
- 每日備忘錄
- 快速範本（常用行程一鍵建立）
- 本週 / 本月完成統計
- 一鍵清理已完成的舊行程
- 桌面提醒通知
- 提醒時間：準時、5 分鐘前、10 分鐘前、30 分鐘前
- 重複行程：每天、每週、每月、每隔 N 天、只工作日（週一～週五）、每月第幾個週幾（例如每月第 2 個週二，含「最後一個」選項）
- 重複行程每日獨立完成，不會全部日期一起打勾
- 行程衝突提醒
- 拖曳行程調整日期；同日拖曳可調整排序
- 搜尋、分類、狀態、重要度篩選
- 今日待辦模式
- 桌面小工具模式
- 關鍵字標籤
- 完成音效
- 每週目標
- 今日重點 3 件事
- 今日完成率統計
- 習慣追蹤與連續天數
- 深色 / 淺色模式
- 自動儲存到瀏覽器本機資料
- 完整 JSON 備份 / 還原
- 匯出 Excel CSV
- 匯出 PDF / 列印
- 台灣國定假日標示（2025–2027年，日／週／月檢視皆會標紅並顯示假日名稱）
- 近 7 天完成統計長條圖
- 番茄鐘 / 專注計時（可綁定今日行程，結束會播放提示音並跳出通知）
- 匯出 / 匯入 .ics（iCal）標準行事曆檔，可與其他行事曆軟體交換行程
- 農曆顯示（日檢視標題與週／月檢視日期格皆顯示農曆日期，內建 1900–2100 年換算，可於「文字設定」開關）
- 日檢視「清單／時間軸」切換：時間軸模式左側顯示小時刻度，行程依開始／結束時間排成方塊，可拖曳方塊下緣調整結束時間；重疊行程會自動並排並標示「⚠️衝突」。選擇會記住在本機設定中。
- PWA（漸進式網頁應用）：可安裝到手機主畫面或電腦桌面，離線也能開啟基本功能。
- 系統推播通知：可另外開啟瀏覽器的系統通知權限，行程提醒時間到會跳出系統通知（有安裝 PWA 時會透過 Service Worker 發送，未安裝則用瀏覽器原生通知），沒有權限或不支援時自動退回畫面右下角提示。
- 雲端同步（Supabase）骨架：預設**未設定**，此時 App 與純本機版完全一樣，不會有任何網路連線。設定好 `config.js` 後，可用 Google 帳號登入，手動或自動把備份資料同步到自己的 Supabase 專案，供多台裝置共用；last-write-wins（後寫入覆蓋），詳細設定步驟見 `CLOUD_SETUP.md`。
- 日／週／月檢視一鍵清除：「🗑 清空當日」（日檢視）、「🗑 清空本週」（週檢視）、「🗑 清空本月」（月檢視）分別只在對應檢視顯示，一鍵清除目前檢視範圍內的所有行程出現次數。非重複行程會直接刪除；重複行程不會刪整個系列，只會把牽涉到的日期標記為「不出現」（`excludedDates`），其他天不受影響。點擊後會先跳出確認彈窗顯示筆數，確認才會清除，且此動作無法復原。

## 使用方式

1. 雙擊 `index.html`。
2. 按「＋ 新增行程」。
3. 輸入事項、日期、時間、分類、提醒。
4. 儲存後會自動保存在本機瀏覽器。
5. 側邊欄「自訂分類」可新增分類名稱與顏色。
6. 上方「文字設定」可修改文字與日檢視時間範圍。
7. 行程卡片按「📍 / 📌」可置頂或取消置頂。
8. 行程卡片按「⧉」可複製到明天。
9. 每日備忘錄會依日期自動儲存。
10. 側邊欄「快速範本」可一鍵建立常用行程。
11. 可用「重要度」下拉篩選高 / 中 / 低優先事項。
12. 上方「今日待辦」只顯示今天未完成行程。
13. 上方「小工具模式」切換精簡畫面。
14. 新增行程時可輸入 `#標籤`，搜尋可找到標籤。
15. 側邊欄「每週目標」可記錄本週重點。
16. 勾選完成會播放短提示音。
17. 上方「清理舊行程」可刪除已完成的過去行程。
18. 上方「備份」可匯出完整資料；「還原」可匯入備份檔。
19. 日／週／月檢視中，紅色標示的日期為台灣國定假日，會顯示假日名稱小字。
20. 側邊欄「完成率」區塊下方會顯示最近 7 天每天完成行程數的長條圖。
21. 上方「🍅 番茄鐘」可開啟專注計時器，預設專注 25 分、休息 5 分，可調整分鐘數並選擇綁定今日某個行程，倒數結束會播放提示音並跳出通知。
22. 新增／編輯行程的「重複」欄位選擇「每隔 N 天」「只工作日」「每月第幾個週幾」時，會出現對應的間隔天數、星期、第幾個等子欄位。
23. 上方「匯出 .ics」可將所有行程輸出成標準 iCal 檔（重複行程會轉成 RRULE）；「匯入 .ics」可讀取外部行事曆的 .ics 檔並轉成行程，無法解析的事件會被跳過並提示筆數。
24. 日檢視標題右側、週／月檢視每個日期格會顯示小字農曆日期；可在「文字設定」勾選「顯示農曆」開關顯示。
25. 切到「日」檢視時，上方會多一組「清單／時間軸」切換鈕。選「時間軸」會依開始／結束時間畫成時間軸方塊，滑鼠移到方塊下緣按住拖曳可調整結束時間，放開即自動存檔；同一時段有多筆行程會並排顯示並標示「⚠️衝突」。方塊上可直接勾選完成，點方塊本體可開啟編輯視窗。
26. 上方「🔔 開啟通知」可要求瀏覽器的系統通知權限；開啟後行程提醒、番茄鐘結束會跳出系統通知，而不只是頁面右下角提示。若瀏覽器不支援或被使用者封鎖，會自動改用頁面內提示，不會出現錯誤。
27. 上方「☁️ 雲端同步」：沒設定 `config.js` 時按下去只會提示「未設定」並請參考 `CLOUD_SETUP.md`。設定好後可按「使用 Google 登入」，登入成功會顯示帳號 email，並提供「立即同步」按鈕與「自動同步」開關（開啟後存檔會在你停止操作幾秒後自動同步）；「登出」只清除本機登入狀態，不影響任何行程資料。
28. 切到「日」檢視時，上方會多一個「🗑 清空當日」按鈕，點擊會提示目前這一天有幾筆行程並確認是否清除；確認後非重複行程會整筆刪除，重複行程只會排除當天（不影響其他天），並跳出提示「已清除今天 N 筆行程」。
29. 切到「週」檢視時，上方會出現「🗑 清空本週」按鈕，統計本週（週一到週日）7 天內所有行程出現次數並確認後清除，邏輯與「清空當日」相同，只是範圍擴大成整週。
30. 切到「月」檢視時，上方會出現「🗑 清空本月」按鈕，統計目前檢視月份（1 號到月底，不含月曆格子補齊的上下月天數）所有行程出現次數並確認後清除，邏輯同上。
31. 上方導航列的「📅」按鈕（或直接點中間的日期標題）會打開年月日選擇器，直接跳到任何一天，不用再用「‹ ›」一格一格翻；週／月檢視會跳到包含那一天的那一週／那個月。新增／編輯行程視窗的「日期」欄位，點一下整個欄位就會打開年月日選擇器。
32. 手機版（螢幕寬度 760px 以下）排版改過：行事曆本體排在最上面，側邊欄（今日重點／完成率／每週目標／習慣追蹤／自訂分類／快速範本）、搜尋篩選區、每日備忘錄都改成收合式面板，預設只顯示標題列，點標題即可展開／收回。畫面右下角新增一個圓形「＋」懸浮按鈕，可直接快速新增行程。工具列只保留常用按鈕（檢視切換、清空當日／本週／本月、主題、☁️ 雲端同步、今日待辦），其餘工具（🔔 開啟通知、備份、還原、文字設定、🍅 番茄鐘、小工具模式、清理舊行程、匯出 Excel、匯出 PDF、匯出／匯入 .ics）收進工具列最後的「⋯ 更多」按鈕，點下去會另外彈出視窗列出這些功能。新增／編輯等視窗在手機上會貼齊畫面底部彈出。此排版變動僅在手機寬度生效，桌面版畫面與操作方式完全不受影響。
33. 側邊欄新增「⏳ 倒數日」面板（在「今日重點」下方）：新增／編輯行程時勾選「⏳ 顯示在倒數日」，就會列在這裡，依「還有 N 天｜標題｜日期」由近到遠排序，重複行程會自動算出下一次出現的日期，當天到期顯示「就是今天！」，點任一列可直接開啟編輯視窗。沒有勾選任何行程時顯示「尚無倒數事項」。
34. 檢視切換多了第四顆「列表」（Agenda 檢視）：從目前日期起算未來 30 天，依日期分組列出所有行程（今天／明天會特別標示），完全沒行程的日期會自動跳過不顯示；上一頁／下一頁會以 30 天為單位前後移動。30 天內都沒有行程時會顯示提示文字。
35. 主題按鈕（🌙／☀️／🌗）改成三段循環：淺色 → 深色 → 自動（跟隨系統）→ 淺色。選「自動」時會依作業系統目前的深色模式設定即時切換，系統設定改變時畫面也會跟著變，不需要重新整理頁面。每次切換都會用提示訊息顯示目前套用的主題。
36. 月檢視的日期格新增淡淡的「熱力圖」底色：當天行程越多，底色越深（1–2 筆、3–4 筆、5 筆以上各一階），滑鼠移到格子上會顯示「N 筆行程」；「今天」的外框與國定假日的紅色標示都不受影響，深色模式下也看得清楚。
37. 「☁️ 雲端同步」對話框新增「🕘 雲端備份版本」（選用功能，需另外執行 `schema-history.sql`）：每次同步推送到雲端時會自動額外留存一份快照，最多保留最新 10 份，點開可看到「時間｜N 筆行程」清單，選一份按「還原」會套用到本機並重新同步成雲端最新版本，用於誤覆蓋時的救援；沒有執行過 `schema-history.sql` 時會提示需要先設定，不影響原本的雲端同步功能。詳見 `CLOUD_SETUP.md` 的「版本備份（選用）」段落。
38. 新增行程視窗的「事項名稱」欄位支援中文自然語言快速輸入：輸入含日期／時間語彙的文字（例如「明天下午3點 開會」「週五 15:30-17:00 會議」「7月20日 晚上八點 聚餐」），欄位失焦（或按下儲存）時會自動解析出日期／開始／結束時間並填入對應欄位，標題自動剝離掉已解析的語彙、只留下事項本身，並跳出「已解析：M/D HH:MM」提示。支援今天／明天／後天／大後天、下週一～下週日（含「下周」寫法）、週一～週日／星期一～星期日／禮拜一～禮拜日（本週已過則算下週）、N月N日／N月N號、N/N（今年，已過則算明年）等日期語彙；以及上午／早上／下午／晚上＋點鐘（可帶半點、分鐘，支援中文數字一～十二）、純數字點鐘（12 以下且無上下午詞時，小於 8 視為下午/晚上）、24 小時制時間（含區間如 `3:30-5:00`）等時間語彙，全形／半形數字與冒號皆可辨識。設計保守：像「3點檔」「第3點」這類非時間用法會被判斷成斷詞失敗而不解析，完全不影響原本輸入；欄位下方有一行小字提示範例用法。純解析邏輯在 `app.js` 的 `parseNaturalDateTime(text, baseDate)`（不依賴 DOM 的純函式）。
39. 「事項名稱」欄位旁新增 🎤 語音輸入按鈕：瀏覽器偵測不到語音辨識 API（`SpeechRecognition`／`webkitSpeechRecognition`）時按鈕會直接隱藏，不影響其他功能。支援的瀏覽器點一下開始聆聽（按鈕變 🔴，提示「聆聽中…再按一次停止」），辨識結果會接在原有文字後面，並自動觸發自然語言解析（見上一點）；再點一次可停止聆聽。辨識失敗或麥克風權限被拒絕時會提示「無法使用麥克風」並還原成 🎤。
40. 上方新增「📊 週回顧」按鈕（手機版收進「⋯ 更多」）：開啟後顯示本週（週一～週日）總行程數、已完成數與完成率（大字）；與上週完成率的比較與增減箭頭（↑／↓／→）；各分類本週完成／總數（最多列出 6 個本週有資料的分類）；下週已排定行程數與前 3 筆日期＋標題預覽；本週每週目標的完成情況。完成判定沿用既有 `completedDates` 與 `occursOnDate()`，重複行程每天各算一次出現。統計計算邏輯在 `app.js` 的 `computeWeeklyReview(baseDate)`（不碰 DOM 的純函式），渲染另外處理。
41. 背景推播提醒（Web Push，**進階選用功能**，預設完全停用不影響任何現有功能）：在雲端同步的基礎上，就算把 App 整個關閉，行程提醒時間到也能收到系統通知。需要另外完成一輪設定（產生 VAPID 金鑰、Supabase 建表、部署 Edge Function、設定排程），詳細教學見 `CLOUD_PUSH_SETUP.md`；沒有設定 `config.js` 的 `webPushPublicKey` 時，「☁️ 雲端同步」對話框不會出現這個功能的任何按鈕或文字。

## PWA 安裝與離線使用

想把「桌面行程表」安裝成手機主畫面 App 或電腦桌面應用程式，必須透過本機伺服器開啟，**不能**直接雙擊 `index.html`（`file://` 協定下瀏覽器不允許註冊 Service Worker，無法安裝、也無法離線使用）。

### 手機加入主畫面

1. 在電腦上雙擊 `start-pwa-local.bat`，它會啟動一個本機網頁伺服器（預設埠 8765）並自動開啟瀏覽器。
2. 讓手機與電腦連上「同一個區域網路（Wi-Fi）」。
3. 到電腦的「命令提示字元」輸入 `ipconfig` 查詢電腦的區域網路 IP（例如 `192.168.1.10`）。
4. 在手機瀏覽器輸入 `http://192.168.1.10:8765/index.html`（IP 換成你電腦的實際位址）。
5. Android（Chrome）：點右上角選單，選「加入主畫面」或「安裝應用程式」。
6. iPhone（Safari）：點下方「分享」圖示，選「加入主畫面」。
7. 加入後即可像 App 一樣從主畫面開啟，之後開啟過一次就能離線使用（資料仍各自存在該裝置瀏覽器的 localStorage，不會自動跨裝置同步）。

### Android 快捷選單（長按主畫面圖示）

安卓（Chrome）加入主畫面後，在主畫面「長按」App 圖示會彈出快捷選單，提供三個捷徑，點了直接開啟對應功能，不用先進主畫面再點按鈕：

- 「新增行程」：直接開啟新增行程視窗。
- 「今日待辦」：直接切換到今日待辦模式。
- 「番茄鐘」：直接開啟番茄鐘計時視窗。

這個功能需要透過網址開啟過一次 App（`https://calendar88.pages.dev/` 或本機伺服器）並加入主畫面才會出現；iPhone（Safari）PWA 目前不支援長按快捷選單。

### 電腦安裝

1. 雙擊 `start-pwa-local.bat` 啟動本機伺服器並開啟 `http://127.0.0.1:8765/index.html`。
2. 在 Edge 或 Chrome 網址列右側點「安裝應用程式」圖示即可安裝成桌面獨立視窗。

### 找不到 Python 時

`start-pwa-local.bat` 需要電腦已安裝 Python（`py` 或 `python` 指令）才能啟動本機伺服器。若沒有安裝，仍可直接雙擊 `index.html` 使用一般功能，但無法安裝 PWA、也無法離線使用。

## 注意事項

- 資料存在目前瀏覽器的 localStorage。
- 若清除瀏覽器資料，行程也會被清除。
- 建議定期使用「備份」匯出 JSON 檔。
- 「匯出 Excel」適合查看；「備份」才適合完整還原。
- 「.ics 匯出／匯入」用於與其他行事曆軟體（如 Outlook、Google 日曆）交換行程，與 JSON 備份是各自獨立的功能，互不取代；還原資料請務必使用「備份／還原」。
- 桌面通知需要瀏覽器允許通知權限；請按上方「🔔 開啟通知」授權，被封鎖時請至瀏覽器網站設定重新開啟。
- 農曆換算為前端內建演算法，涵蓋西元 1900–2100 年，超出範圍不會顯示農曆文字。
- 日檢視「時間軸」模式的拖曳調整結束時間僅影響該筆行程的 `end` 欄位，不會改變開始時間；若拖曳後結束時間早於開始時間會自動取消變更。
- PWA／離線快取只在透過本機伺服器（`http://` 或 `https://`）開啟時才會啟用；直接雙擊 `index.html`（`file://`）仍可使用全部行程功能，只是不會安裝、不會離線快取。
- 雲端同步預設停用，`config.js` 沒填 Supabase 網址與金鑰時 App 完全不會發出網路請求。啟用步驟、安全性說明（anon key 為何可以公開、資料靠 RLS 保護）與常見問題請見 `CLOUD_SETUP.md`；資料表定義在 `schema.sql`。登入用的權杖只存在瀏覽器本機，**不會**被包進「備份」匯出的 JSON 檔。Google 登入導回流程需要透過 `http://` 網址（例如 `start-pwa-local.bat` 啟動的本機伺服器），直接雙擊 `index.html` 的 `file://` 模式無法完成登入。
